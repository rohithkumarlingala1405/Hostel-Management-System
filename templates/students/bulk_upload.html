📋 CSV Format Guide

Required columns:

name roll_no

Optional columns:

email block room_number

Valid Block values:

B1 — Boys Block 1 B2 — Boys Block 2 B3 — Boys Block 3 G1 — Girls Block 1

Example CSV:

name roll_no email block room_number
Ravi Kumar 20241001 ravi@iitgoa.ac.in B1 101
Priya Sharma 20241002 priya@iitgoa.ac.in G1 101
Amit Patel 20241003

📌 Notes:
• Column names must match exactly (lowercase)
• Duplicate roll numbers will be skipped automatically
• Both block AND room_number must be provided to assign a room
• Valid blocks: B1, B2, B3 (Boys) and G1 (Girls)
• If block/room not found or room is full, student is added without a room
• Rows with missing name or roll_no are skipped
